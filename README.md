## 🔱 About KEDOL

**KEDOL** is the native utility token powering the **Kedolik Ecosystem**, built on the Solana blockchain.

Designed with transparency, security, and real utility in mind, KEDOL supports decentralized finance applications, ecosystem incentives, staking rewards, and platform benefits.

KEDOL is not just a token — it is the foundation of a growing Web3 ecosystem.
